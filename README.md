# Soda
個人的な興味から始めた、現在も個人サーバーで運用しているDiscordアプリです。本格的にプログラミングを学び始めたきっかけであり、何もわからない状態から独学で作り続けたPythonプロジェクトです。

## 使用ツール
Python, 2026年よりコーディングエージェントGeminiを使用。
主な使用ライブラリ：discord.py, chromadb, langchain, onnxocr

## 開発期間
2023.03.19 ~

## 工夫した点・苦労した点
- Discordを利用する間に感じた「こういった機能が欲しい」から、複数の機能を考え、調べ、実装しました
- イベント情報、チャット機能、コミュニティツール、個別連絡機能、画像認識、リマインダー、ツイッター連携、テキスト読み上げなどの機能を開発しました
- Pythonの入門時からずっと作り続けていたため、拡張性や保守性の改善に苦労しました

## 課題点・改善点
- 最初から一人でずっと作り続けていたため、可読性を考慮しておらず、チーム制作に移行できない
